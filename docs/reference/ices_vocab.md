# ICES controlled vocabularies (code lists)

ICES publishes controlled vocabularies (also called **code lists**:
tables of allowed values like "Gear" codes) via a public REST API.

## Details

These are not OWL ontologies; use them for categorical fields and
reporting codes, not for ontology IRIs.
